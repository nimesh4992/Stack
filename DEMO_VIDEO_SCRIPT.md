# Demo Video Script for Google Play Store Review

## Video Requirements
- **Duration:** 2-3 minutes
- **Format:** YouTube (unlisted)
- **Resolution:** 1080p minimum
- **Audio:** Clear narration or text overlay

---

## Video Script

### Scene 1: App Introduction (0:00 - 0:15)
**Action:** Show app icon and open app
**Narration/Text:**
> "HabitFinance is a personal finance management app that helps users track expenses and build financial habits."

---

### Scene 2: Core Functionality Without SMS (0:15 - 0:45)
**Action:** 
1. Navigate to Home screen
2. Tap FAB (+) button
3. Manually add a transaction
4. Show transaction in list

**Narration/Text:**
> "Users can manually add transactions at any time. SMS auto-detection is an optional convenience feature."

---

### Scene 3: Navigate to Settings (0:45 - 1:00)
**Action:**
1. Tap Profile tab
2. Navigate to Settings
3. Scroll to "SMS Auto-Detection" section

**Narration/Text:**
> "SMS auto-detection can be enabled in Settings."

---

### Scene 4: Prominent Disclosure (1:00 - 1:45)
**Action:**
1. Toggle "Auto-Read Bank SMS" switch ON
2. **SHOW FULL DISCLOSURE SCREEN**
3. Slowly scroll through the disclosure content
4. Highlight key sections:
   - What we access
   - How data is used
   - What we DON'T do
   - Privacy assurance

**Narration/Text:**
> "Before any permission is requested, users see a detailed disclosure explaining exactly what data will be accessed and how it will be used."

**IMPORTANT:** Show the user must scroll to bottom before Accept is enabled.

---

### Scene 5: User Consent (1:45 - 2:00)
**Action:**
1. Show "Accept & Continue" button
2. Tap Accept
3. Android permission dialog appears

**Narration/Text:**
> "Only after explicit consent does the Android permission dialog appear."

---

### Scene 6: Feature in Action (2:00 - 2:30)
**Action:**
1. Show SMS received (simulated bank SMS)
2. Show transaction detected notification
3. Show transaction in pending review
4. Approve transaction

**Narration/Text:**
> "When a bank SMS is received, it's processed locally on the device and presented for user review."

---

### Scene 7: Disable Option (2:30 - 2:45)
**Action:**
1. Return to Settings
2. Toggle SMS reading OFF
3. Show confirmation

**Narration/Text:**
> "Users can disable SMS reading at any time from Settings."

---

### Scene 8: Privacy Assurance (2:45 - 3:00)
**Action:**
1. Navigate to Privacy Policy screen
2. Scroll through SMS section
3. Show "Your data never leaves your device" badge

**Narration/Text:**
> "All data remains on the user's device. No SMS content is ever transmitted to any server."

---

## Key Points to Emphasize

1. **SMS is OPTIONAL** - App works fully without it
2. **Prominent Disclosure** - Shown BEFORE system permission
3. **On-Device Processing** - No data leaves device
4. **User Control** - Can disable anytime
5. **Transparency** - Clear explanation in Privacy Policy

---

## Recording Tips

1. Use Android device (not emulator for SMS demo)
2. Record in portrait mode at 1080p+
3. Clear notifications before recording
4. Ensure stable internet (for Play Console access)
5. Use screen recording with audio or add text overlays
6. Upload as UNLISTED YouTube video
